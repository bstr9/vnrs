# 仪表盘首页 + 右侧监控面板 开发计划

## 整体布局

```
┌─────────────────────────────────────────────────────────────────┐
│  菜单栏                                              账户余额  │
├──────┬────────────────────────────────────────────────────────────┤
│      │  仪表盘首页 (默认Tab)                                     │
│ 交易 │  ┌─────────────┐ ┌─────────────┐ ┌─────────────┐          │
│ 面板 │  │  账户总览    │ │  今日盈亏    │ │  风险监控    │          │
│ 250px│  └─────────────┘ └─────────────┘ └─────────────┘          │
│      │  ┌─────────────────────┐ ┌─────────────────────┐          │
│      │  │  持仓概览            │ │  策略状态           │          │
│      │  └─────────────────────┘ └─────────────────────┘          │
│      │  ┌─────────────────────┐ ┌─────────────────────┐          │
│      │  │  今日盈亏曲线        │ │  系统状态           │          │
│      │  └─────────────────────┘ └─────────────────────┘          │
│      │  切换Tab后进入详细视图: 行情/委托/成交/K线/策略/回测       │
├──────┴────────────────────────────────────────────────────────────┤
│  底部: 日志 / 资金 / 持仓                                        │
└─────────────────────────────────────────────────────────────────┘
```

- 仪表盘是默认首页，启动后第一个看到全局概览
- 右侧监控面板(后续)无论哪个Tab都可见，可折叠
- 点击卡片/行可跳转到对应详细Tab

## 卡片详细设计

### 卡片1: 账户总览
- 总资产(大字)、可用余额、保证金占用(进度条)
- 点击 [查看详情] → 跳转底部"资金"Tab
- 数据来源: AccountMonitor

### 卡片2: 今日盈亏
- 盈亏金额+收益率(绿色/红色)
- 盈利交易笔数/亏损交易笔数
- 胜率、盈亏比
- 点击 → 跳转"成交"Tab
- 数据来源: TradeMonitor 过滤今日

### 卡片3: 风险监控
- 保证金风险率弧形仪表/进度条
- 风险等级: <50%安全(绿) / 50-70%正常(黄) / 70-85%警告(橙) / >85%危险(红+闪烁)
- 警戒线标注
- 数据来源: 账户保证金+持仓计算

### 卡片4: 持仓概览
- 迷你表格: 品种/方向/持仓/均价/盈亏
- 盈亏用颜色编码(绿盈红亏)
- 默认按盈亏绝对值降序
- 合计浮动盈亏
- 点击行 → 打开该品种K线图
- 点击 [查看全部] → 底部"持仓"Tab

### 卡片5: 策略状态
- 策略名/状态灯/今日盈亏
- 状态: 🟢运行中 / 🟡已初始化 / 🔴已停止
- [全部启动] [全部停止] 批量操作
- 点击行 → 跳转"策略"Tab
- 数据来源: StrategyPanel

### 卡片6: 今日盈亏曲线
- 按时间(9:00~当前)累计盈亏折线图
- 零线高亮，当前点标注
- 最高/最低/当前值
- 数据来源: 按分钟/小时聚合今日成交

### 卡片7: 系统状态
- 网关连接状态(🟢已连接+延迟 / 🟡重连中 / 🔴断开)
- 最近3条通知(策略触发/风控/连接)
- 点击 [查看全部] → 通知中心
- 数据来源: MainEngine网关状态

## 响应式布局

### 宽屏 >1600px
- 第一行: 3列卡片(账户/盈亏/风险)
- 第二行: 持仓(4列) + 策略(2列)
- 第三行: 曲线(4列) + 系统(2列)

### 中等 1200-1600px
- 第一行: 3列卡片
- 第二行: 持仓(全宽)
- 第三行: 策略 + 系统
- 第四行: 曲线(全宽)

### 窄屏 <1200px
- 所有卡片单列堆叠

## 数据刷新策略

| 数据 | 刷新频率 | 来源 |
|------|----------|------|
| 账户余额 | 实时 | AccountMonitor |
| 持仓盈亏 | 实时 | PositionMonitor + Tick |
| 策略状态 | 实时 | StrategyPanel |
| 今日盈亏 | 按需计算 | TradeMonitor过滤 |
| 网关状态 | 实时 | MainEngine |
| 盈亏曲线 | 每分钟 | 聚合计算 |

## 实现顺序

- [ ] 1. Dashboard基础框架 - CentralTab::Dashboard + 卡片容器 + 响应式布局
- [ ] 2. 账户总览卡片 - 数据来自AccountMonitor
- [ ] 3. 今日盈亏卡片 - 过滤今日Trade + 计算胜率/盈亏比
- [ ] 4. 风险监控卡片 - 保证金风险率 + 进度条 + 等级颜色
- [ ] 5. 持仓概览卡片 - 迷你持仓表 + 点击跳转
- [ ] 6. 策略状态卡片 - 策略状态灯 + 批量操作
- [ ] 7. 今日盈亏曲线 - 累计盈亏折线图
- [ ] 8. 系统状态卡片 - 网关连接 + 通知历史
